# 07 — خارطة التوسّع | Scalability Roadmap

## المرحلة 1 — الإطلاق (MVP) ✅ (الحالي)
واجهة فاخرة + كتالوج + سلة + طلب واتساب + مُصمّم ستائر مجاني + حاسبة + لوحة تحكم + SEO. توصيل لكل العراق، دفع عند الاستلام.

## المرحلة 2 — البيانات الحيّة
- نقل مصدر المنتجات من `lib/data` إلى **Supabase** (نفس الأنواع — تغيير دالة الجلب فقط).
- رفع صور حقيقية إلى `product-images`.
- تفعيل حفظ الطلبات في جدول `orders` عبر `/api/orders` ومتابعتها من اللوحة.

## المرحلة 3 — الدفع الإلكتروني (ماستر كارد)
- إضافة Route Handler لبوّابة دفع (مثل Areeba/بوّابة عراقية أو Stripe دولياً).
- حالة طلب «مدفوع» + إيصال. القيم جاهزة (`payment_method: 'mastercard'`).

## المرحلة 4 — رفع جودة الـAI (اختياري ومدفوع)
- تفعيل مزوّد عبر `AI_PROVIDER` (Replicate/Stability/Flux/OpenAI) لـInpainting واقعي.
- إبقاء الوضع المجاني للمتصفح كخيار افتراضي. (راجع `08-AI-VISUALIZER.md`.)

## المرحلة 5 — النمو
- **فروع متعددة**: حقل `branch` في المنتجات/الطلبات + فلترة حسب الفرع.
- **B2B / جملة**: قوائم أسعار للمقاولين + طلب عرض سعر.
- **تطبيق موبايل**: المشروع PWA الآن (`manifest`)؛ يمكن تغليفه أو بناء تطبيق يستهلك نفس Supabase.
- **تعدّد العملات/الدول**، **نقاط ولاء**، **كوبونات** (جدول `coupons` جاهز).
- **تحليلات**: ربط Google Analytics / Meta Pixel عند الحاجة.

## ملاحظات أداء عند الكبر
- صفحات ثابتة + ISR (إعادة توليد دورية) للمنتجات.
- CDN لصور Supabase + أحجام `next/image`.
- فهارس قاعدة البيانات (مضافة على `category`,`featured`,`status`,`created_at`).
- Supabase ينمو أفقياً؛ ترقية الخطة عند زيادة الحمل.
