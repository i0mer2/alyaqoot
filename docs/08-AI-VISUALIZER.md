# 08 — وحدة معاينة الستائر بالذكاء الاصطناعي | AI Visualizer

## الفلسفة: «شبه واقعي + مجاني تماماً» (حسب طلبك)
المعاينة الافتراضية تتم **داخل متصفح الزبون** عبر **Canvas** — بدون أي API ولا تكلفة، وبخصوصية تامة (صورة الغرفة لا تُرفع لأي خادم). وفي نفس الوقت بُنيت **طبقة خدمة معيارية** تسمح بالترقية لمعاينة فوتوغرافية واقعية عبر مزوّد مدفوع لاحقاً — دون تغيير الواجهة.

## كيف يعمل الوضع المجاني (Client Composite)
الملف: [`src/lib/visualizer/draw.ts`](../src/lib/visualizer/draw.ts) + الواجهة [`src/components/visualizer/RoomVisualizer.tsx`](../src/components/visualizer/RoomVisualizer.tsx)

1. الزبون يرفع صورة غرفته (أو يستخدم غرفة وهمية افتراضية).
2. ترسم الخوارزمية الستارة فوق الصورة مع:
   - **كسرات (folds)** عبر تدرّج لوني متكرّر.
   - **لمعان وظلال** لإحساس القماش.
   - **نقشة** (سادة / مخطّط / دمشقي) و**شفافية** (تول).
   - تحكّم كامل: اللون، التغطية (جانبين/كامل/يسار/يمين)، العرض، الطول، الموضع، كثافة الكسرات.
3. **مقارنة**: التقاط عدة لقطات لأنماط مختلفة وعرضها جنباً إلى جنب.
4. **حفظ** الصورة (PNG) أو **الطلب مباشرة** (واتساب/سلة).

> «شبه واقعي» مناسب جداً للاختيار بين الألوان والأنماط قبل الشراء، وسريع (فوري) ومجاني حتى لو الجودة ليست فوتوغرافية ١٠٠٪.

## طبقة الخدمة المعيارية
```
src/lib/ai/
├── types.ts                 # عقد موحّد (VisualizeRequest/Result/Provider)
├── service.ts               # getProvider() + visualize() حسب AI_PROVIDER
└── providers/
    ├── browser.ts           # الوضع المجاني (افتراضي)
    └── paid-stubs.ts        # Replicate / Stability / OpenAI (جاهزة للتفعيل)
```
نقطة الـAPI: [`src/app/api/ai/visualize/route.ts`](../src/app/api/ai/visualize/route.ts).

## كيف تُفعّل مزوّداً واقعياً لاحقاً (اختياري)
1. أضِف المفتاح في البيئة، مثلاً:
   ```
   AI_PROVIDER=replicate
   REPLICATE_API_TOKEN=...
   ```
2. في `providers/paid-stubs.ts` استبدل جسم الدالة بنداء نموذج **Inpainting/img2img** (مثل SDXL-Inpaint أو Flux):
   - أرسل صورة الغرفة + قناع منطقة النافذة + وصف الستارة (اللون/الخامة).
   - استقبل رابط الصورة الناتجة وأعِده في `imageUrl` مع `mode:'image'`.
3. الواجهة ستعرض الصورة المولّدة بدل التركيب المحلي تلقائياً (نفس العقد).

## أفكار لتحسين الواقعية لاحقاً
- **كشف النافذة تلقائياً** (تجزئة) لمطابقة المنظور والإضاءة.
- **تطابق الإضاءة**: تحليل لون/سطوع الغرفة وتعديل ظلال الستارة.
- **حفظ المشاريع** للمستخدمين المسجّلين (جدول `saved_designs`).

## التكلفة
- الآن: **صفر** (كل شيء في المتصفح).
- عند التفعيل المدفوع: تُحسب لكل توليد صورة حسب المزوّد — تبقى اختيارية.
